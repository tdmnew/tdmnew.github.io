This Site!

This site was my attempt at coming to grips with HTML, CSS and JavaScript following my introduction having taken the CS50 course at EDX. I wanted to create a website that made use of transition animations between pages.

Unfortunately, for a first attempt, there a lot of redundancy in my code (As you will see by checking the page source). My hope was that the code could be easily repurposed into another site just by changing a few of the images and element names. Maybe I’ll clean up the code a little and allow it to be repurposed – although it’s more likely I’ll get bored and make a new site once I find a new framework to play with!
